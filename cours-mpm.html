La méthode MPM (Méthode des Potentiels Métra) expliquée en détail


1. Présentation de la méthode MPM

La MPM (ou méthode des potentiels) a été développée par Bernard Roy en 1958. En MPM :

  • Les tâches sont représentées par des nœuds (sommets du graphe)
  • Les contraintes de succession sont représentées par des arcs avec une valeur (délai)
  • Elle permet de gérer des contraintes de type FD, DD, FF
  • Elle s'appuie sur le calcul des rangs, dates au plus tôt et dates au plus tard

2. Structure du nœud MPM

NOM DE LA TÂCHE
Date + tôt
tôt
Date + tard
tard

Structure standard d'un nœud dans le graphe MPM


3. Calcul des dates au plus tôt (phase aller)

Règle de calcul (parcours AVANT) :

On part du nœud début (date = 0) et on progresse vers la droite :

Date au plus tôt (j) = MAX [ Date au plus tôt (i) + délai (i→j) ]

On prend le maximum car la tâche ne peut démarrer qu'après toutes ses prédécesseurs.


4. Calcul des dates au plus tard (phase retour)

Règle de calcul (parcours ARRIÈRE) :

On part du nœud fin (date + tard = date + tôt) et on remonte vers la gauche :

Date au plus tard (i) = MIN [ Date au plus tard (j) − délai (i→j) ]

On prend le minimum pour ne pas décaler les successeurs.


5. Marges et chemin critique

Marge totale

MT = Tard(i) − Tôt(i)

Retard max sans retarder le projet

Marge libre

ML = Min[Tôt(j)] − Tôt(i) − d(i)

Retard max sans affecter les successeurs

🔴 Chemin critique = ensemble des tâches avec MT = 0

Ce chemin relie le nœud début au nœud fin et correspond à la durée totale minimale du projet.


💡 À RETENIR

En MPM, les tâches sont les nœuds. Le chemin critique est la chaîne de tâches dont les marges totales sont nulles. La phase aller donne les dates au plus tôt (MAX), la phase retour donne les dates au plus tard (MIN).

Exercice 1 Niveau : Fondamental

Calcul MPM — Projet de construction

Contexte : Un projet de construction comprend les tâches suivantes :

Tâche Durée (j) Contrainte
Début0—
A — Terrassement5Début FD 0
B — Fondations8A FD 0
C — Livraison matériaux3Début FD 0
D — Gros œuvre10B FD 0, C FD 0
Fin0D FD 0
Questions :
  1. Dessinez le graphe MPM avec les nœuds
  2. Calculez les dates au plus tôt (phase aller)
  3. Calculez les dates au plus tard (phase retour)
  4. Identifiez le chemin critique et les marges
Phase aller (dates au plus tôt) :
  • Début : tôt = 0
  • A : tôt = 0 + 0 = 0 → finit à j.5
  • C : tôt = 0 → finit à j.3
  • B : tôt = max(tôt_A + dur_A) = 0 + 5 = 5 → finit à j.13
  • D : tôt = max(B+0, C+0) = max(13, 3) = 13 → finit à j.23
  • Fin : tôt = 23 + 0 = 23 jours
Phase retour (dates au plus tard) :
  • Fin : tard = 23
  • D : tard = 23 − 10 = 13
  • B : tard = 13 − 8 = 5
  • C : tard = 13 − 3 = 10
  • A : tard = 5 − 5 = 0
  • Début : tard = min(A→0, C→0) = 0
Marges totales :
  • A : MT = 0 − 0 = 0 (critique)
  • B : MT = 5 − 5 = 0 (critique)
  • C : MT = 10 − 0 = 10 jours
  • D : MT = 13 − 13 = 0 (critique)
Chemin critique : Début → A → B → D → Fin (23 jours)
Exercice 2 Niveau : Intermédiaire

MPM avec contraintes variées — Projet informatique

Contexte : Un projet de développement logiciel présente des contraintes non standards :

ArcTypeValeur (j)Signification
Début → AFD0A démarre dès le début
A → BDD2B démarre 2j après le début de A
A → CFD0C démarre à la fin de A
B → CFF3C finit 3j après la fin de B

Durées : Début=0j, A=6j, B=4j, C=5j, Fin=0j

Question : Calculez les dates au plus tôt pour chaque tâche.
Rappel des formules selon le type de contrainte :
  • FD (Fin→Début) : tôt(j) ≥ tôt(i) + durée(i) + délai
  • DD (Début→Début) : tôt(j) ≥ tôt(i) + délai
  • FF (Fin→Fin) : tôt(j) + durée(j) ≥ tôt(i) + durée(i) + délai → tôt(j) ≥ tôt(i) + durée(i) + délai − durée(j)
Calcul :
  • Début : tôt = 0
  • A : tôt = 0 + 0 = 0
  • B (DD+2 de A) : tôt ≥ tôt(A) + 2 = 0 + 2 = 2
  • C (FD de A) : tôt ≥ 0 + 6 = 6
  • C (FF+3 de B) : tôt ≥ tôt(B) + dur(B) + 3 − dur(C) = 2 + 4 + 3 − 5 = 4
  • C : tôt = max(6, 4) = 6
  • Fin : tôt = 6 + 5 = 11 jours
